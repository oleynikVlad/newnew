# AnonBoard — Анонімна платформа

Сучасний веб-застосунок для анонімного обговорення об'єктів (людей, компаній, речей тощо).

## Технології

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Prisma ORM** + SQLite
- **Zod** для валідації

## Функції

- Додавання об'єктів з фото (URL або upload)
- Категорії: людина, компанія, річ, інше
- Система тегів
- Пошук та сортування (нові / популярні / обговорювані)
- Вкладені коментарі (threaded)
- Лайки/дизлайки
- Кнопка "поскаржитися" з автоматичним приховуванням
- Фільтрація токсичних слів
- Rate limiting
- Адмін-панель
- Темна тема

## Встановлення та запуск

```bash
# Встановити залежності
npm install

# Створити базу даних та застосувати міграції
npx prisma migrate dev

# Заповнити тестовими даними
npx tsx prisma/seed.ts

# Запустити dev-сервер
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)

## Структура проєкту

```
src/
├── app/
│   ├── api/
│   │   ├── entities/     # CRUD для об'єктів
│   │   ├── comments/     # Коментарі
│   │   ├── reports/      # Скарги
│   │   ├── votes/        # Голосування
│   │   ├── upload/       # Завантаження файлів
│   │   ├── tags/         # Теги
│   │   └── admin/        # Адмін API
│   ├── entity/[id]/      # Сторінка об'єкта
│   ├── add/              # Додавання об'єкта
│   ├── admin/            # Адмін-панель
│   ├── layout.tsx
│   ├── page.tsx          # Головна сторінка
│   └── globals.css
├── lib/
│   ├── prisma.ts         # Підключення до БД
│   ├── moderation.ts     # Фільтрація контенту
│   ├── rate-limit.ts     # Обмеження запитів
│   └── validators.ts     # Zod-схеми валідації
prisma/
├── schema.prisma         # Схема бази даних
└── seed.ts               # Тестові дані
```

## API Endpoints

| Метод | Шлях | Опис |
|-------|------|------|
| GET | `/api/entities` | Список об'єктів (пошук, сортування, фільтр) |
| POST | `/api/entities` | Створити об'єкт |
| GET | `/api/entities/[id]` | Деталі об'єкта |
| POST | `/api/comments` | Додати коментар |
| POST | `/api/reports` | Поскаржитися |
| POST | `/api/votes` | Голосувати |
| POST | `/api/upload` | Завантажити фото |
| GET | `/api/tags` | Список тегів |
| GET | `/api/admin` | Статистика адмін-панелі |
| POST | `/api/admin` | Дії модерації |
